# Who Benefits from AI?
Data visualization project exploring global AI inequality.

## Research Question
Which factors influence the global distribution of AI – and which countries risk being left behind?

## Datasets
- `AI-index.xlsx` — Oxford Insights Government AI Readiness Index
- `GDP_pro_Kopf_Laender.csv` — World Bank GDP per capita
- `GDP_Laender_Metadaten.csv` — World Bank country metadata
- `share-of-individuals-using-the-internet.csv` — ITU Internet access
- `private-investment-in-artificial-intelligence.csv` — Our World in Data
- `UNESCO_UIS_WIDEF.csv` — UNESCO Education Data
- `HDR25_Statistical_Annex_HDI_Table.xlsx` — UNDP Human Development Index
